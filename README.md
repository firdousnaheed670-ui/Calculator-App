# # 🧮 Calculator Web App

A simple calculator web application built using **Python (Flask)** and **Bootstrap**.
It allows users to perform basic arithmetic operations like addition, subtraction, multiplication, and division.

---

## 🚀 Live Demo

## 📌 Features

* Add two numbers
* Subtract numbers
* Multiply numbers
* Divide numbers
* Handles division by zero
* Clean and responsive UI
